# ResearchChatAI

**Design and run chatbot research with full control.**

ResearchChatAI is a fully customizable chatbot platform that allows researchers to easily set up and conduct studies with AI agents — no coding experience required. Customize every aspect of your agent, collect clean and replicable data, and get up and running in minutes.

ResearchChatAI is hosted at [researchchatai.com](https://researchchatai.com/). The source code provided here can also be installed on local or self-hosted environments.

## How It Works

1. **Create a Study** — Sign up for a free researcher account and set up your first study
2. **Configure Your Agent** — Define study characteristics, enable conditions, and control AI agent behavior, name, avatar, and instructions
3. **Collect & Analyze** — Distribute your study to participants and analyze their responses

## Key Features

- **No Coding Required** — Intuitive interface for designing and managing studies
- **Fully Customizable** — Control every aspect of your AI agent and study design
- **Multiple Conditions** — Set up experimental conditions and participant variables
- **End-to-End Encryption** — All message texts are encrypted, ensuring only you have access
- **Data Export** — Download conversations and participant submissions for analysis
- **Open Source** — Community-driven development and full transparency
